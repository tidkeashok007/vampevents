    Vamp Events!

    This Python Django project is an online shopping website. It utilizes Django's powerful features, including Models and ORM for the SQLite database, Views and URL routing for handling requests, Templates for dynamic HTML pages, Forms and Validation for user input, Authentication and Authorization for secure access, and Django Admin for backend management. The project offers a seamless shopping experience while leveraging SQLite for efficient data storage and retrieval.

    Django Admin

    The Django Admin interface is utilized to manage the e-commerce website's backend. It provides an intuitive administration panel to perform tasks such as adding , deleting or updating products

    Authentication and Authorization

    Django's built-in authentication system is employed to handle user registration, login, and logout functionalities. User authentication ensures secure access to user-specific features and protects sensitive information.

    Templates

    Django's template engine is employed to create dynamic HTML pages, facilitating the rendering of data from the backend to the frontend. This allows for the display of product listings, shopping cart contents, and order details in a visually appealing manner.

    Static and Media Files

    Django manages static files, such as CSS and JavaScript, to enhance the website's appearance and functionality. It also handles media files, including product images, allowing them to be uploaded and served efficiently.

    Models and ORM

    Django's ORM(Object-Relational Mapping) is used to define the database structure and interact with the SQLite database. It provides a convenient way to create, retrieve, update, and delete records, allowing seamless integration with the e-commerce website.

    Paytm Payment Integration

    Integrated the Paytm payment gateway, which involved implementing various payment options for seamless and secure transactions. The primary objective was to enhance the user experience by providing a reliable and widely used payment solution.
